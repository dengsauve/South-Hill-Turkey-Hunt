# South Hill Turkey Hunt
Hunt the turkeys, save thanksgiving, feed the community, and rid the south hill of those pesky turkeys in this zork-inspired short story.

## Download and Play

### Pre-requisites
- [ ] Python 3 installed

### Download

1. Choose a following method to get the code
   1. clone the repository (recommended)
   2. [Download the ZIP](https://github.com/dengsauve/South-Hill-Turkey-Hunt/archive/refs/heads/main.zip) and extract.
2. Open the repository's directory in terminal
3. Run `python3 src/main.py`
4. Follow the directions, and use `help` or `hint`

### Scoring
Golf is the name of the game. Complete the objectives and save the day in the fewest number of moves. 

Alternatively, set a record for the most moves. Live your best life.
